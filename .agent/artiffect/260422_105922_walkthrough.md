## 🛠️ 코드(Code) 구현 내역

### (옵션 8) JSON-RPC Batching 구성
- **대상 파일**: `src/blockchain/service/blockchain-rpc.service.ts`
- **구현 상세**:
  - `ethers.JsonRpcProvider` 초기화 구문에 옵션 객체를 전달했습니다.
  - `batchMaxCount: 50`: 50개의 개별 RPC 쿼리(`getBlockByNumber` 등)가 발생하는 순간, 이를 50번의 개별 HTTP 요청으로 분산하지 않고 **단 1개의 HTTP POST 페이로드 배열**에 묶어 노드로 한 번에 전송하게 됩니다.
  - `staticNetwork: true`: 요청마다 노드의 `chainId`가 변경되었는지 확인하는 기본 검증 루틴을 우회하여 불필요한 노드 트래픽을 아예 차단했습니다.
